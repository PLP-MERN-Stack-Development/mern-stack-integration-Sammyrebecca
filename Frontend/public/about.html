About This Blog App

This is a small MERN-stack blog application used for learning and integration exercises.

What you'll find here

  • A React frontend served from Frontend/
  • An Express + MongoDB backend served from Backend/
  • Support for creating posts, uploading images, and user authentication (JWT)

Notes

Because this file is in the public/ folder it will be served statically by the React dev server and available at /about.html.

To view this page while the frontend dev server is running, open:

http://localhost:3000/about.html

If you want this page to be part of React routing (no full reload), I can add a React About component and wire up a route instead.
